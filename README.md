E-Lunch
======================
Random display today's lunch!!

## URL
https://www.e-lunch.info/

### 構成
- Nginx
- Node.js
- TypeScript
- Vue.js
- Webpack
- Socket.io
- dotenv
- jQuery

## ローカル開発環境構築
```
$ npm run build:dev
$ npm run dev
```
